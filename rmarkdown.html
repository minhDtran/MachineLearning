1.Project description : This is the machine learning project using the Rpart method from the caret function in the R studio.

The program reads two excel cvs files. One is the training data set, and the other is testing data set. The outcome variable in the training data set is classe , and the rest are the predictor of the training data set. I will partitition the training data set into training data set and testing data set. I will design a prediction model for the training set, and use it to test against the test data set. I will record the accurancy and their confidence interval. I then use the prediction model to test against the real testing data set.
